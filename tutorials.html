Tutorials for Modular Psychophysics (mPsy)¶

Installation¶

If you haven’t downloaded Modular Psychophysics yes, click on this link.

The core of Modular Psychophysics requires only Python to run. One other essential dependency (pyglet) is bundled with the package. However, we also suggest installing Numerical Python (numpy). It is very likely that even the simplest experiment will require some numerical computation. Some of the stimuli we use in tutorials use Python’s Numpy module.

If you are completely new to Python, we suggest installing the free version of Enthought Python Distribution recently rebranded as Canopy Express. For Mac OS, this distribution is one of the few reliable ways to obtain all scientific packages (including matplotlib) and make them work without further effort.

Getting Started¶

Although mPsy’s core is written in Python, it can be controlled from other lanuguages. Working examples of controlling mPsy presentation remotely are available for the following languages:

  • Python - Psychophysics in “pure” Python
  • MATLAB - Control from MATLAB
  • R - Controlling mPsy experiments from R
