Stimulus Library¶

Dot Lattice¶

Dot lattice stimulus is an array of dots, dots are organized along two orientations with constant within orientation inter-dot distances.

Dots in this stimulus can be redered as gratings, so that each dot can have its own sptial frequency and orientation.

Automatically generated stimulus parameters for "stimuli.DotLattice" will be here.

Random Dot Kinematograms¶

Random dot kinematograms (RDK) as visual stimuli composed of randomly scattered and moving points. Commonly used RDK stimulus is composed of two subgroups of points, each can be controlled independently in order to study motion perception of coherent groups in noisy (in-coherent) backgrounds.

This stimulus displays only a single group, multiple groups can be obtain by overlaying multiple instances of ths stimulus.

Automatically generated stimulus parameters for "stimuli.RDK" will be here.

Tiled Image¶

Tiled image is a stimulus object that enables the presentation of images. Images can be shown unmodified, scaled, rotated, or on tiles obtained by splitting the image by a regular grid. Some properties of tiles can be controlled independently.

Automatically generated stimulus parameters for "stimuli.Tiles" will be here.

Grating¶

One of the most elementary vision stimulus, periodic grating with periodically changing contrast. The contrast can be controled (modulated) in space and time. This stimulus has a built in gaussian mask, which can be used as a circular aperture.

Automatically generated stimulus parameters for "stimuli.Grating" will be here.

Cube¶

Cube arbitrarily rotated and scaled in three dimensions. The colors of vertices and faces can be controlled.

Automatically generated stimulus parameters for "stimuli.Letters" will be here.

Alphabet Grid for BCI Spelling¶

A grid of letters commonly used in Brain Computer Interface (BCI) research. Letter of alphabet are shown in 6x6 grid, cells of this grid can be periodically highlighted. This stimulus sends events on each update of stimulus, so they can be logged for ERP analysis.
